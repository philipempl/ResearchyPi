## Getting started

```shell
sudo apt-get update
sudo apt-get upgrade

sudo apt install python3 idle3
sudo apt-get install python3-pip

sudo apt-get install git
git clone https://github.com/philipempl/ResearchyPi.git
cd ./ ResearchyPi

sudo pip3 install -r requirements.txt
python3 ./setup.py

python3 ./main.py
```
